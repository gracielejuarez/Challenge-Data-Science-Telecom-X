📊 Telecom X – Análise de Evasão de Clientes (Churn)

📌 Descrição do Projeto

Este projeto tem como objetivo analisar os dados da empresa Telecom X para identificar fatores que influenciam a evasão de clientes (churn).

A empresa enfrenta um alto índice de cancelamentos e precisa compreender os padrões associados à perda de clientes. A partir dessa análise exploratória, a equipe de Data Science poderá desenvolver modelos preditivos e estratégias para reduzir o churn.

O projeto foi desenvolvido em Python no Google Colab, aplicando técnicas de ETL e Análise Exploratória de Dados (EDA).

🎯 Objetivos

Coletar dados a partir de uma API.

Realizar o processo de ETL (Extração, Transformação e Carga).

Tratar inconsistências e valores ausentes.

Explorar os dados para identificar padrões de evasão.

Criar visualizações que apoiem a tomada de decisão.

Gerar insights estratégicos sobre o comportamento dos clientes.

🛠️ Tecnologias Utilizadas

Python

Pandas

NumPy

Matplotlib

Requests

Google Colab

📂 Etapas do Projeto

1️⃣ Extração de Dados

Os dados foram coletados a partir de uma API disponibilizada para o projeto e convertidos para um DataFrame utilizando Python.

2️⃣ Transformação dos Dados

Nesta etapa foi realizado o tratamento dos dados, incluindo:

Conversão de tipos de dados

Padronização de variáveis

Tratamento de valores nulos

Ajustes para garantir a consistência das informações

3️⃣ Análise Exploratória (EDA)

Foram realizadas análises e visualizações para identificar padrões relacionados ao churn, como:

Distribuição da evasão de clientes

Relação entre churn e cobrança mensal

Impacto do tipo de contrato

Influência de serviços adicionais

📊 Principais Insights

A taxa de churn identificada foi de aproximadamente 26,5%, indicando um nível relevante de evasão.

Clientes com contrato mensal apresentam maior probabilidade de cancelamento.

Valores mais altos de cobrança mensal estão associados a maiores taxas de churn.

Clientes sem serviços adicionais de suporte ou segurança tendem a cancelar com maior frequência.

🚀 Próximos Passos

Construção de modelos preditivos de churn.

Identificação de clientes com maior risco de cancelamento.

Desenvolvimento de estratégias de retenção.

📎 Conclusão

A análise exploratória permitiu identificar fatores relevantes associados à evasão de clientes. Esses insights servem como base para o desenvolvimento de modelos de Machine Learning e estratégias de retenção, contribuindo para reduzir o churn na Telecom X.
